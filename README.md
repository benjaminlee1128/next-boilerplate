### Getting started

Run the following command on your local environment:

```shell
git clone https://github.com/arrofirezasatria/nextjs-mui-boilerplate.git my-project-name
cd my-project-name
npm install
```

Then, you can run locally in development mode with live reload:

```shell
npm run dev
```
